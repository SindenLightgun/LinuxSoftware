# Installation

## Install

Install Sinden Lightgun dependencies, utilities, and borders

```
cd /home/pi; \
git clone path/to/repo; \
chmod 755 setup*; \
./setup-lightgun.sh; \
./setup-retropie.sh; \
```

## Per Architecture

Determine which architecture you are running on and move the files to
your home directory

- Pi-Arm
- x86-32bit
- x86-64bit

```
cp -r arch/YOUR-ARCHITECTURE/* ../bin/
```
